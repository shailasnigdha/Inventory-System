📦 All Products API

View all products across all sellers in the system.

Endpoint:
GET /admin/api/products

Returns a JSON list of all products with details including name, price, quantity, seller information, and timestamps.

📋 All Orders API

View all orders in the system with complete details.

Endpoint:
GET /admin/api/orders

Returns a JSON list of all orders including buyer, seller, product, quantity, price, and status information.

👥 All Users API

View all registered users in the system.

Endpoint:
GET /admin/api/users

Returns a JSON list of all users including username, email, roles, and account status.
