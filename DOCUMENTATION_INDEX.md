### Key Endpoints
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/dashboard/buyer` | GET | Buyer dashboard page |
| `/auth/me` | GET | Get current user (NEW) |
| `/auth/register/{role}` | POST | Register user |
| `/auth/login` | POST | Login |
| `/products` | GET | Browse products |
| `/orders` | POST | Place order |
| `/orders/buyer/{id}` | GET | Get buyer orders |
